Sentiment Analysis of Song Lyrics Using VADER and NLTK


NLTK (Natural Language Toolkit) is a comprehensive library for working with human language data (text). It includes tools for text processing, classification, tokenization, parsing and more.


VADER (Valence Aware Dictionary and sEntiment Reasoner) is a specific sentiment analysis tool included within the NLTK library. It is used to analyze the sentiment of text (whether it's positive, negative, or neutral) and provides a sentiment score.


The stacked bar chart illustrates the distribution of song sentiments across different years. Each bar represents a year, with the segments within the bar showing the breakdown of sentiment labels such as positive, negative, neutral . The chart provides a visual overview of how the sentiment associated with songs has evolved over time.


The bubble chart visualizes the average sentiment scores of songs by different artists from 2014 to 2024. The chart plots the sentiment scores against the years, with each bubble representing an artist. The size of the bubble indicates the number of songs released by the artist in a given year, and the color corresponds to each artist. This allows for a comparative view of both the sentiment trends and the volume of songs released by each artist over the years.


Note: This data focuses specifically on the top 15 songs listened to each year between 2014 and 2024.



These bar charts display the top 15 songs with the strongest sentiment scores , both negative and positive. The sentiment scores are plotted on the x-axis, with negative values indicating negative sentiment and positive values reflecting positive sentiment. The chart uses a red color scale for negative sentiment, where darker shades represent stronger negativity, and a green color scale for positive sentiment, where darker shades represent stronger positivity. This visualization highlights the songs with the most pronounced emotional tones, whether negative or positive.


Positive Sentiment: When a song's lyrics convey positive emotions like happiness, love, or hope, it results in a positive sentiment score.

Negative Sentiment: When the lyrics express negative emotions such as sadness, anger, or frustration, it leads to a negative sentiment score.
